The Wordpress tool of the Watchizzle TV widget.  

== Description ==

Plug-in adds the 'Recently Recorded' Watchizzle TV widget to your sidebar. Displays 'Recently Recorded' and 'What are Watchizzle users Recording?' posts. Allows you to click on a link and watch it instantly.

== Installation ==

Follow steps described here:

1. Extract contents of archive
2. Upload contents of Watchizzle TV folder to the 'wp-content/plugins' folder
3. Log in to the WordPress admin and go to the 'Plugins' section. You should see Watchizzle TV in the list of available plugins
4. Activate plugin by clicking 'activate' link
5. To appear in the sidebar go to the "Appearance" section, and select the "Widgets"-subsection.
6. You can position the widget within the sidebar by simply dragging and dropping the 'Watchizzle TV'-entry.
